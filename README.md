# Tela de Login

Este projeto é um modelo de tela de login estática, desenvolvida com HTML e CSS.

## Sobre

O projeto foi criado para fins de aprendizado, demonstrando como estruturar e estilizar um formulário de login moderno. Ele inclui campos para nome de usuário e senha, bem como botões de login social simulados.

## Como Usar

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` no seu navegador.
3. Preencha os campos e clique em "Continue" para ver a página de dashboard simulada.

## Tecnologias

- HTML5
- CSS3

## Créditos

Código modificado a partir de um original open source, utilizado apenas para fins educacionais.
